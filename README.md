## 项目介绍

基于 Spring Boot + MyBatis + MySQL + COS + WebSocket 的 **智能素材库**。

项目网址：http://150.158.89.168/

功能介绍：

1）所有用户都可以在平台公开上传和检索图片素材，快速找到需要的图片。可用作表情包网站、设计素材网站、壁纸网站等：
